Default Style
  • White cards, light gray borders
  • Minimal shadows — barely there
  • Small circular connection handles
  • Neutral colors throughout
  • No color unless user opts in via style.color
Node Differentiation
  • All nodes are rounded rectangles
  • Decision: small icon badge, side handles
  • Start: subtle green dot indicator
  • Sections: stacked in a light gray band
  • Type indicated by content, not shape
Customization
  • Colors are opt-in, not default
  • style.color tints border + icon
  • style.variant for filled/outlined/ghost
  • Theme prop for light/dark
  • CSS custom properties for full override
